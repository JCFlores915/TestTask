## Clonar el repositorio

```bash
git clone https://github.com/JCFlores915/TestTask.git
```

## Instalar dependencias

```bash
yarn install
```

## Ejecutar el proyecto

```bash
yarn start
```

## Correr los test

```bash
yarn run test:watch
```
